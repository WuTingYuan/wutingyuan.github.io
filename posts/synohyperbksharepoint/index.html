Synology Hyper-Backup 備份到商務用 OneDrive

2023-01-23
#Synology  #Hyper Backup  #OneDrive  #OneDrive for Business 

Synology 的備份軟體 Hyper Backup 原生沒有支援備份到商務用 OneDrive，想要備份就得另外想辦法，網路上找了一下，發現有解決方案， 是透過 WebDAV 加上在 Docker 中架設 Auth Proxy 的方式連接到商務用 OneDrive 做備份，簡單設定後就能備份了🥳

1. 部署 SharePoint 的 WebDAV 代理 #

1-1. 透過 Docker 指令建立 # 


    $ docker run \
    --name sharepoint-proxy \
    -d \
    -p 3000:3000 \
    -e PROXY_TARGET=https://example-my.sharepoint.net/ \
    --restart always \
    skleeschulte/basic-to-sharepoint-auth-http-proxy:v0.0.1

1-2. 使用 docker Compose 建立 # 


    version: "3"
    services:
        container_name: basic-to-sharepoint-auth-http-proxy
        image: skleeschulte/basic-to-sharepoint-auth-http-proxy:v0.0.1
        environment:
          - "PROXY_TARGET=https://example-my.sharepoint.net/"
        restart: always
        ports:
          - target: 3000
            published: 3000
            protocol: tcp

2. 建立備份任務 #

備份目的地：WebDAV
伺服器位址：http://127.0.0.1:3000/personal/account_name/Documents
使用者帳號：backup@wkopiaspupn.onmicrosoft.com
密碼：<password>
資料夾：輸入「伺服器位址」、「使用者帳號」、「密碼」後點擊會自動帶出
目錄資料夾：選擇後自動帶出
